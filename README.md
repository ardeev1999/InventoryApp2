# Inventory App

Мобильное приложение для инвентаризации техники и мебели. 
Позволяет импортировать данные из Excel, сканировать QR-коды и штрих-коды, отслеживать статус предметов.

## Функциональность
- Импорт из .xlsx, .xls, .csv в SQLite.
- Просмотр списка предметов с фильтрацией по статусу (найден/отсутствует).
- Сканирование QR-кодов и штрих-кодов через камеру.
- Автоматическое изменение статуса при совпадении инвентарного номера.
- Отдельный список найденных предметов.

## Технологии
- Kotlin
- SQLite
- Библиотеки для работы с Excel (например, Apache POI)
- Камера и сканирование (ML Kit / ZXing)
